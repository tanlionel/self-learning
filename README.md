## 🗺️ Knowledge Pillars (The Chapters)

### 📂 [Chapter 1: DSA & Logic (LeetCode)](./chapters/01-dsa)
*Goal: Sharp logic and optimized problem-solving.*
* **Priority Patterns:** Sliding Window, DFS/BFS, Dynamic Programming, and Topological Sort.
* **Complexities:** Deep understanding of Time/Space trade-offs in distributed data processing.

### 📂 [Chapter 2: Java Ecosystem Mastery](./chapters/02-java)
*Goal: Becoming a Java expert (Version 8 through 21).*
* **The Core:** Stream API optimization, Optional usage, and Record types.
* **Deep Dive:** JVM Memory Model (Heap/Stack), Garbage Collection tuning (G1/ZGC).
* **Concurrency:** From `Thread` and `synchronized` to `CompletableFuture` and Virtual Threads.

### 📂 [Chapter 3: Spring Ecosystem](./chapters/03-spring)
*Goal: Building robust, enterprise-grade Microservices.*
* **Spring Boot:** Internal workings of Auto-configuration and custom Starters.
* **Persistence:** Advanced JPA (Entity Graphs, L2 Cache) and Transaction Isolation levels.
* **Security:** JWT, OAuth2, and securing communication between microservices.

### 📂 [Chapter 4: Kubernetes & DevOps](./chapters/04-k8s)
*Goal: Infrastructure as Code and Scalability.*
* **Containerization:** Multi-stage Docker builds and image size optimization.
* **Orchestration:** K8s Objects (Deployments, Services, Ingress, ConfigMaps, Secrets).
* **Observability:** Distributed tracing, Prometheus metrics, and ELK logging.



### 📂 [Chapter 5: System Design & Architecture](./chapters/05-system-design)
*Goal: Thinking in distributed systems.*
* **Scaling:** Horizontal vs. Vertical, Database Sharding, and Read Replicas.
* **Caching:** Multi-level caching strategies (Local Cache + Redis).
* **Communication:** REST vs. gRPC vs. Message Brokers (Kafka/RabbitMQ).
